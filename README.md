# studio-ghibli-films
Phase 1 Project

These Ruby Gems provide a CLI to view a list of Studio Ghibli films along with information for each film.

# Installation
Type the code below.

    $ bundle install

# Usage
Type the code below and follow the on screen prompts.

    $ ./bin/studio-ghibli-films
