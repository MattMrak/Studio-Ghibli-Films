# best-gardens-around-the-world
Phase 1 Project

These Ruby Gems provide a CLI to view 50 of the best gardens around the world according to the House Beautiful website.

# Installation

    $ bundle install

# Usage

Type the code below and follow the on screen prompts.

    $ best-gardens-around-the-world
